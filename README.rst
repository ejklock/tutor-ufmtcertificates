ufmtcertificates plugin for `Tutor <https://docs.tutor.overhang.io>`__
===================================================================================

Installation
------------

::

    pip install git+https://github.com/ejklock/tutor-ufmtcertificates.git

Usage
-----

::

    tutor plugins enable ufmtcertificates
